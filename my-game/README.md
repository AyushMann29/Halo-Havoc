# Halo Havoc - Client Frontend

This directory contains the frontend code for **Halo Havoc**, built with **React**, **Vite**, and **p5.js**.

## 📌 Main Project Documentation
For complete documentation of the game mechanics, tech stack, architecture, live metrics, and setup instructions, please refer to the **[Main Root README](../README.md)**.

## 🚀 Quick Start (Client Only)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
